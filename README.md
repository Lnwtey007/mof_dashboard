🇹🇭 Thai
การประยุกต์ใช้ Machine Learning สำหรับการกักเก็บก๊าซมีเทนใน MOFs เพื่อการใช้งานในยานยนต์
คลังข้อมูล (Repository) นี้รวบรวมระบบการวิเคราะห์ข้อมูลและแดชบอร์ดสำหรับคัดกรองวัสดุโครงข่ายโลหะอินทรีย์ (MOFs) จากฐานข้อมูล ARC-MOF โดยมีจุดมุ่งหมายเพื่อค้นหา MOF ที่มีประสิทธิภาพสูงสุดในการกักเก็บก๊าซมีเทน ผ่านการวิเคราะห์คุณสมบัติทางโครงสร้าง (เช่น ขนาดรูพรุน) ร่วมกับการสกัดค่าประจุของโลหะ (Partial Metal Charge) จากไฟล์ CIF โปรเจกต์นี้แสดงผลข้อมูลในรูปแบบภาพเพื่อค้นหา "จุดที่เหมาะสมที่สุด (Sweet Spot)" สำหรับการดูดซับมีเทน เพื่อต่อยอดการพัฒนาเทคโนโลยีกักเก็บพลังงานทางเลือกสำหรับยานยนต์

🇬🇧 English
Machine Learning for Methane Storage in MOFs for Vehicular Applications
This repository contains a data analysis pipeline and an interactive dashboard for screening Metal-Organic Frameworks (MOFs) from the ARC-MOF database. It aims to identify the optimal MOFs for methane storage by analyzing geometric properties (like pore size) and extracting partial metal charges from CIF files. The project visualizes the "sweet spot" for methane uptake to support advanced vehicular storage applications.

🇷🇺 Russian
Машинное обучение для хранения метана в MOF для транспортных средств
В этом репозитории представлен конвейер анализа данных и интерактивная панель управления для отбора металлоорганических каркасов (MOF) из базы данных ARC-MOF. Проект направлен на поиск оптимальных MOF для хранения метана путем анализа геометрических свойств (таких как размер пор) и извлечения частичных зарядов металлов. Проект визуализирует «идеальные условия» для поглощения метана, чтобы поддержать разработку передовых систем хранения для транспортных средств.
